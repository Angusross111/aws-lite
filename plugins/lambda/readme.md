# `@aws-lite/lambda`

> Official `aws-lite` plugin for Lambda

> Maintained by: [@architect](https://github.com/architect)


## Install

```sh
npm i @aws-lite/lambda
```

Optionally install types:

```sh
npm i -D @aws-lite/lambda-types
```
