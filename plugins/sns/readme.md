# `@aws-lite/sns`

> Official `aws-lite` plugin for SNS

> Maintained by: [@architect](https://github.com/architect)


## Install

```sh
npm i @aws-lite/sns
```

Optionally install types:

```sh
npm i -D @aws-lite/sns-types
```
