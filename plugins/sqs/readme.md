# `@aws-lite/sqs`

> Official `aws-lite` plugin for SQS

> Maintained by: [@architect](https://github.com/architect)


## Install

```sh
npm i @aws-lite/sqs
```

Optionally install types:

```sh
npm i -D @aws-lite/sqs-types
```
