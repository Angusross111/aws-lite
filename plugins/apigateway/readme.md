# `@aws-lite/apigateway`

> Official `aws-lite` plugin for API Gateway

> Maintained by: [@architect](https://github.com/architect)


## Install

```sh
npm i @aws-lite/apigateway
```

Optionally install types:

```sh
npm i -D @aws-lite/apigateway-types
```
